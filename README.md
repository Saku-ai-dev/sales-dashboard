# 売上管理ダッシュボード

## 概要

架空の売上データを可視化したWebアプリです。月別売上グラフ・サマリーカード・一覧テーブルをHTML単一ファイルで実装しています。

## 使用技術

- HTML / CSS / JavaScript
- Chart.js（CDN）

## 機能

- 月別売上の棒グラフ表示
- 年間総売上・月平均・最高月のサマリー表示
- 月別売上一覧テーブル（前月比表示）
- レスポンシブ対応

## 使い方

`index.html` をブラウザで開くだけで動作します。

```bash
# ファイルをブラウザで直接開く
open index.html
```
